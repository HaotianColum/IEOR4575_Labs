## Working on the Lab
Assuming you are at the root of this directory:
```
$ conda activate ieor4575
$ wandb login
$ jupyter notebook
```
Then, complete problems 1 and 2 found in lab1_problem1.ipynb and lab1_problem2.ipynb, respectively.
